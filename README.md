# 🛍️ ZenithMart – Modern Multi-Vendor E-Commerce Platform

ZenithMart is a modern full-stack e-commerce platform that enables customers to discover, explore, and purchase premium products across multiple categories. The platform provides secure authentication, product management, shopping cart functionality, order processing, role-based dashboards, and an elegant responsive user experience for both customers and administrators.

# 🚀 Setup Instructions

## 1. Clone the Repositories

```bash
git clone https://github.com/kazij317-code/zenithmart-client

git clone https://github.com/kazij317-code/zenithmart-server
```

---

## 2. Install Dependencies

### Client

```bash
cd zenithmart-client
npm install
```

### Server

```bash
cd zenithmart-server
npm install
```

---

## 3. Configure Environment Variables

Create a `.env.local` file inside the client project.

Create a `.env` file inside the server project.

---

## 4. Run Development Servers

### Client

```bash
npm run dev
```

### Server

```bash
npm start
```

---

# 🔑 Environment Variables

## Client (.env.local)

```env
NEXT_PUBLIC_API_URL=http://localhost:5000

NEXT_PUBLIC_BASE_URL=http://localhost:3000
```

---

## Server (.env)

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

CLIENT_URL=http://localhost:3000
```
